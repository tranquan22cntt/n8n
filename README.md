# n8n Automation Landing Page

Landing page giới thiệu dịch vụ tự động hóa quy trình bằng n8n, được thiết kế theo phong cách SaaS dark UI, có bảng giá, so sánh gói, lợi ích, FAQ và form liên hệ kết nối Google Sheets.

## Demo

- Website: [n8n-eight-pink.vercel.app](https://n8n-eight-pink.vercel.app/)

## Nội Dung Chính

- Header phong cách n8n với menu điều hướng và nút liên hệ tư vấn.
- Hero section nền tối, hình ảnh lightning, khói màu và vòng sáng chuyển động.
- Bảng giá dịch vụ gồm Starter, Professional, Business và Enterprise.
- Bảng so sánh tính năng giữa các gói.
- Khu lợi ích nổi bật với icon và card hiện đại.
- FAQ dạng accordion.
- CTA/form liên hệ có gửi dữ liệu về Google Sheets.
- Responsive cho desktop, tablet và mobile.

## Công Nghệ

- `HTML5` cho cấu trúc trang.
- `CSS3` cho giao diện, animation và responsive.
- `JavaScript` cho FAQ, scroll reveal, mobile menu, form submit và load dữ liệu bảng giá từ Google Sheets.
- `Google Apps Script` làm API trung gian cho Google Sheets.
- `Vercel` để deploy website từ GitHub.
